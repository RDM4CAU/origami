# Arbeitsphase II: Create Metadata Schema

**“Design Your Own Origami Metadata Schema”**

Zeitbedarf (20 Minuten)

**Hinführung/Problemstellung (2 Minuten)**

Eine internationale Origami-Community möchte eine offene, durchsuchbare Datenbank aufbauen.

Ziel: Modelle langfristig auffindbar, zitierbar und reproduzierbar machen.

Eure Aufgabe: Entwickelt ein Metadatenschema.

Gruppenarbeit (10 Minuten)

Welche Felder sollen verpflichtend sein?

Welche optional?

Welche kontrollierten Vokabulare braucht ihr?

Was ist das absolute Minimum für Reproduzierbarkeit?

Wie berücksichtigt ihr FAIR?

Nebenbedingung:

Maximal 12 Metadatenfelder insgesamt.

## Reflexion Phase II

- Kernfelder
